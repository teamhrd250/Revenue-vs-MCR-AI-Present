# Deployment Checklist

- [ ] Semua file dan folder diunggah ke root repository GitHub.
- [ ] `app.py` berada di root repository.
- [ ] File Excel berada di folder `data/`.
- [ ] Logo berada di folder `assets/`.
- [ ] Repository tidak berisi `.streamlit/secrets.toml`.
- [ ] Main file path di Streamlit adalah `app.py`.
- [ ] `OPENAI_API_KEY` dimasukkan pada Streamlit App Settings > Secrets.
- [ ] Browser mengizinkan akses mikrofon.
- [ ] Dashboard diuji pada mode Executive dan Smartphone.
